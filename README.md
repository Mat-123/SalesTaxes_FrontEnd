# SalesTaxes - Frontend

Frontend application built with React and Vite.

## Tech Stack

- React
- Vite
- JavaScript
- Fetch API
- React Router

## Requisirement

- Node.js 18+
- npm

## Installation

1) Clone the repository:

git clone https://github.com/username/SalesTaxes_FrontEnd.git
cd SalesTaxes_FrontEnd

2) Install the dependencies:
npm install

3) Start development server:
npm run dev
